# AGENTS.md

This repository may be modified by AI coding assistants and other automated agents.

For every code change, agents must ensure the repository is left in a clean validation state.

Required checks before considering a change complete:

1. `make lint` must pass with no lint warnings or errors.
2. `make test` must pass with no test failures.

Do not leave the codebase with known lint warnings, failing tests, or partially validated changes.
